Function and function expressions
Introduction Return Expression Scope

A function is a block of code that performs a specifi task.

Benefits of using function

  • Function makes the code reusable. You can declare it once and use it multiple times.
  • Function makes the program easier as each small task is divided into a function.
  • Function increases readability.

JavaScript also has a huge number of inbuilt functions.
For example, Math.sqrt() is a function to calculate the square root of a number.

  1. Declaring a function
    function greet1() {
    console.log('Hello there!');
    }
    greet1();    //calling function
  2. Function Expression
    const speak = function() {
    console.log('Have a great day!');
    }
  3. Add two numbers using a function
    function add(a,b) {
    console.log(a+b);
    }
    add(5,2);// 7
    add(8,4);// 12    //calling function
  4. Function with parameters
    function greet2(name) {
    console.log('Hello ' + name + ':');
    }
    let name = prompt('Enter a name: ');
    greet2(name);    //alling function

A parameter is a value that is passed when declaring a function
Note: When a value is passed when declaring a function, it is called Parameter. And when the function is called, the value passed is called argument.

In the above program, the greet function is declared with a name parameter. The user is prompted to enter a name. Then when the function is called, an argument is passed into the function.

Function return

The return statement can be used to return the value to a function call.
The return statement denotes that the function has ended. Any code after return is not executed.

  1. Function return
    function add1(c,d) {
    return c+d;
    }
    //take input from the user
    let number1 = parseFloat(promt('Enter first number: '));
    let number2 = parseFloat(promt('Enter second number: '));
    //alling function
    let result = add1(number1, number2)     //the sum is = first number + second number

    2. In the above program, the sum of the numbers is returned by the function using the return statement. And that value is stored in the result variable.

If nothing is returned, the function returns an undefined value

Function expressions

  1. //function
    let x = function(num) {
    return num*num;
    }
    console.log(x(4));//output = 16
    // can be used as variable value for other variables
    let y = x(3);
    console.log(y);//output = 9

In the above program, variable x is used to store the function. Here the function is treated as an expression.
The function above is called an anonymous funcion.
