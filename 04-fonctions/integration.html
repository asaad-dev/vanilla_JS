Storing the information you need

Most of the time, a JavaScript application needs to work with information. Here are two examples:

  • An online shop - the information might include goods being sold and shopping cart.
  • A chat application - the information might include users, messages, and much more.

Variables are used to store this information.

What is a variable?

A variable is a container for a value, like a number we might use in a sum, or a string that we might use as part of a sentence. But one special thing about variables is that their contained values can change.

We can declare variables to store data by using the var, let, or const keywords

const - is like let, but the cvalue of the variable can't be changed.

Name things right

Talking about variables, there's one more extremely important thing.

A variable naming is one of the most important and complex skills in programming. A quick glance at variable names can reveal which code was written by a beginner versus an experienced developer.

Ternary operator

When you want to execute a block of code if a particular test evaluates to true, you often use the if-else statement
Example, if age is greater than 16, then allow the person to drive can be coded as follws:


            var age = 19;
            var canDrive;
            if (age > 16) {
                canDrive = 'yes';
            } else {
                canDrive = 'no';
            }

In this example you can use ternary opertor as shortcut for the if-else statement as follws:


            var age = 19;
            var canDrive = age > 16 ? 'yes' : 'no';

The code looks much cleaner
In general, the syntax of the ternary operators is as follws:

condition ? expression_1 : expression_2

It's very hard to write any kind of interesting software without making use of conditionals and boolean logic.

Ternary operators can help you write less code, but be careful using them, as they also tend to make your code more difficult to read.
