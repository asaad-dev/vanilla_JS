JS III - 01 Les structures de contrôle

1- Les conditions et les boucles

Il existe deux types de structures de contôle de base, que l'on retrouve dans tous les langauges de programmation, et donc en JS.

  • Les structures de contrôle conditionnelles : Les conditions
  • Les structures de contrôle de boucles : Les boucles

Les conditions vontnous permettre d'éxécuter un certain nombre d'instruction si et seulement si une certaine conditions est vérifiées

Les boucles vont nous permettre d'exécuter un bloc de code en boucle tant qu'une condition donnée est vérifiée.

A- Présentation des conditions

  • La condition if (si);
  • La condition if ...else (Si ... sinon);
  • La condition if ...elseif ... else(si ... sinon ... sinon).

Dans le cas oû la comparaison est vérifiée, le JavaScript renvoie le booléen true. Dans le cas contraire, le booléen false est renvoyé.

Le JavaScript va automatiquement comparer la valeur à gauche de l'Opérateur à celle de droite selon l'Opération fourni et renvoyer le booléen ture si la comparasion est validée ou false si la comparaison ne l'est pas.
