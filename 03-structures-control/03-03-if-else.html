JS III - 03 if else

1- if ...else ...

La structure condiitionnelle if ... else (en français si ... sinon) sera plus compléte car on va exécuter un bloc de code si un test renvoie trueou un autre dans le cas contraire.

If ... els ...

  • if(x > 1) {
    alert('x contient une valeur supérieur à 1');
    }else {
    alert('x contient une valeur inférieur à 1');
    }
