JS III - 05 Les opérateurs Logiques

Ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaison dans nos tests ou d'inverer la valeur logique du résaltat d'un test.

1- Présentation et utilisation des opérateurs logique

Le JS supporte 3 opérateurs logiques : l'opérateur logique "ET", "OU" et "NON"

Les opérateurs logiques
Opérateurs, nom Opérateur, symbole Description
AND(ET) && renvoie true si toutes les comparaison sont évaluée à true, ou false ninon
OR'OU || renvoie true, si au moins l'une des comparaison est évaluée à true, ou false sinon
NO (NON) ! renvoie false si une comparaison est évaluée à true, ou renvoie true dans le cas contraire
