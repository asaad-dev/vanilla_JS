JS III - 04 if else if else

1- if ...else ... if ...else en JS

La condition if ... else if ...else (si ... sinon si ... sinon) est une structure condidionnelle encore plus compléte puisqu'elle permet de générer et de prendre en charge autant de cas que l'on souhaite.

Nous allons pouvoir écrire autant de else if que l'on veut dans cette condition

if(x > 1) {
alert('x contient une valeur strictement supé');
} else if(x == 1) {
alert('x contient la valeur'); } else {
alert('x contient une valeur strictement inférieure à 1');
}

Dés qu'un test va être validé le JS va ignorer les tests suivants ? Il faut donc faire bien attention à l'ordre des else if lors de l'écriture d'une condition pour obtenir le résultat souhaité.
