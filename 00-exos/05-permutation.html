Permutation de 2 variables !

Ecrivez un programme '05-permutation.js' ayant initialement le contenu suivant:

var nomber1 = 5;

var nomber2 = 3;

//Tapez votre code ici(sans rien modiffieer d'autre!)

console.log(nombre1); //Doit afficher 3

console.log(nombre2); //Doit afficher 3

* Á l'endroit indiqué par un conmmantaire, ajoutez le code nécessaire pour inverser les valeurs des deux variables. cf. capture d'écran

Il existe plusieurs solutions à cet exericie.
Astuce : vous n'êtes pas limité(e) à l'utilisation de deux variables
