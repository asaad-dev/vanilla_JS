Valeurs !

Observez le code ci-dessous et tentez de prévoir les valeurs finales de chaque variable à la fin de chaque ligne (par exemple aprés la ligne var a = 2;) si je faisais un console.log de a j'aurai 2 comme résultat. cf. capture d'écran

L'idée est de faire les calculs vous-même, sur une feuille par exemple AVANT de regarder dans la console !

Opération Explication
var a = 2; Explication : (a = 2)
a = a - 1; Explication : (2-1 = 1)
a++; Explication : (1+1 = 2)
var b = 8 Explication : (b = 8)
b += 2 Explication : (8+=2) = 10
var c = a + b * b; Explication : (10x10 = 100) +2 = 102
var d = a * b + b; Explication : (2x10)= 20+10 = 30
var e = a *(b + b); Explication : (10+10) = 20x2 = 40
var f = a * b / a Explication : (2x10) 20 / 2
var g = b / a * a; Explication : (10/2 = 5) *2 = 10
