JS - II 01 introduction

Qu'est-ce qu'une variable ?

Une variable est un conteneur servant à stocker des information de maniére temporaire, du texte (une chaîne de caractéure) ou un number. la variable varie !

Une variable et la valeur sont deux choses différentes

2- Régles de déclaration

On dit (comme en PHP) que l'on "déclare" une variable. On déclare une variable avec les mots-clefs

  • let
  • var
  • Le nom d'une variable doit obligatoirement commencer par une lettre ou un underscore _
  • Le nom d'une variable ne doit contenir que des lettres, des chiffres et des _
  • Il n'y a pas d'espace
  • cf. aller voir || introduction.JS

3- Initialiser une variable

Quand on assigne une variable à une variable vide on dit que l'on stocke une valeur ou qu'on initialise une variable

4- Différences entre "var" et "let"

letest plus récent que var

  • A- La remontée ou hosting des variables

    Avec Var on n'est pas obligé de déclarer la variable avant la manipuler ou de l'affecter

  • B- La redéclaration des variables

    AVec var on avait le droit de déclarer plusieurs fois une même variable en utilisant à chaue fois var (ce qui avait pour effet de modifier la valeur stockée). On ne peut pas le faire avec let

  • C- La portée des variables

    La "portée" d'une variable désigne l'endroit oû cette variable va pouvoir être utilisée dans un script

    Nous reparlerons donc de cette portée des variables lorsque nous aborderons les fonctions en JavaScript.

  • Conclusion

    La syntaxe de déclaration des variables avec let correspond à la nouvelle syntaxe. La syntaxe avec var est l'ancienne syntaxe qui est vouée à disparaitre.

    Vous devriez donc aujourd'hui toujours utiliser le mot clef let pour déclarer vos variables

5- Quelle utilité pour les variables ?

Il sera pratique de stocker différents types de données ou d'informations dans une variable

Exemple avec la fonction prompt()on récupérera une données envoyée par un utilisateur, on fera un script qui contiendra une variable qui stockera la donnée quand elle sera enovoyée.
