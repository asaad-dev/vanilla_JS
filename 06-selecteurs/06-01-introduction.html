  • Dans une variable qui se trouve dans window.onLoad() /// var select1 = titreNiveau1
  • Dans une variable simple var select2 = titreNiveau1;
  • Avec getElementById directement dans le document document.getElementById('titreNiveau1');
  • Avec getElementByClassName je récupére un ensemble une collection de class document.getElementsByClassName('maClass1');
  • Avec getElementsByTagName je récupère un ensemble d'élément par leur nom
