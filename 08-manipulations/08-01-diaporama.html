JS - VIII 01 - Les diaporamas

3 diaporamas en JS

1- Un 1er diaporama, avec des photos de plages...

1ére façon de faire, sur les images directement je mets un onclick...

Dans la balise img, je met cet attribut : onclick="this.src='img/e.jpg';this.alt='une autre plage';"

2- Un 2nd diaporama...

Ce second diaporama est fait avec une fonction anoyme dans un écouteur d'événement, dans les scripts en bas de page

document.getElementById('btnDiapo2').addEventListener('click', function() { document.getElementById('imgDiapo2').src ='img/piscine2.jpg'; document.getElementById('imgDiapo2').alt ='La piscine est belle'; })
